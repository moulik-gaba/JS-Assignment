# JS-Assignment

This repository contains my solutions for Assignment 4 of Fundamentals of Web Design.

## About the Assignment

In this assignment, we were asked to solve 6 problems using basic JavaScript.
All inputs are taken using `prompt()` and outputs are shown using `alert()` as required.

## Files

* Q1.js – Digit Gatekeeper
* Q2.js – Roll-Seed Lock
* Q3.js – Mirror Corridor
* Q4.js – Fare Calculator
* Q5.js – Skipping Numbers
* Q6.js – Contest Score Judge

## How I Solved It

**Q1:**
I used a loop from L to R and checked each number for the given conditions like divisibility, digit sum and making sure it doesn't contain 0.

**Q2:**
I followed the steps given in the question and updated the number 3 times based on whether it is even or odd.

**Q3:**
I tried values of X starting from 0 and checked when the number becomes palindrome and divisible by K.

**Q4:**
I calculated the fare step by step by applying all the conditions given in the question.

**Q5:**
I kept adding numbers while skipping multiples of (seed + 2) until the sum reached at least N.

**Q6:**
I calculated the score using the formula and then applied the conditions to get the final result.
